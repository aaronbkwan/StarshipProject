to run the project please do the following

1. run the start-project.bat
2. head to http://localhost:4200/

if you don't like the idea of running the bat please do the following:
1. open the Star Wars API in visual studio code
2. open the visual studio code terminal
3. cd into StarshipProject
4. run the following command: "dotnet run"
5. open a new terminal in visual studio code
6. cd into ClientApp
7. run the following command: "ng serve"
8. head to http://localhost:4200/
